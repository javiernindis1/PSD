# Deployment
## Tahap Implementasi dan Penyajian Aplikasi

Tahap **Deployment** merupakan tahap akhir dalam metodologi **CRISP-DM**, di mana model dan hasil analisis yang telah dibangun disajikan dalam bentuk aplikasi yang dapat digunakan oleh pengguna.

Pada proyek ini, hasil pemodelan data time series diimplementasikan dalam bentuk **aplikasi web menggunakan Streamlit** dan dipublikasikan secara online.
